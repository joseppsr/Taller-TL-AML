CAPA CUSTOMIZADA
Implementa una capa Keras llamada DebtBurdenLayer.

Debe:
1. Heredar de tf.keras.layers.Layer.
2. Recibir un tensor con 3 columnas en este orden:
   [AMT_INCOME_TOTAL, AMT_CREDIT, AMT_ANNUITY]
3. Calcular:
   annuity_income_ratio = AMT_ANNUITY / (AMT_INCOME_TOTAL + epsilon)
   credit_income_ratio = AMT_CREDIT / (AMT_INCOME_TOTAL + epsilon)
4. Aplicar saturación suave con tanh:
   annuity_ratio_sat = tanh(annuity_income_ratio / annuity_tau)
   credit_ratio_sat = tanh(credit_income_ratio / credit_tau)
5. Usar:
   epsilon = 1e-6
   annuity_tau = 0.5
   credit_tau = 5.0
6. Devolver un tensor de shape (batch_size, 2):
   [annuity_ratio_sat, credit_ratio_sat]
7. Implementar get_config para que la capa sea serializable.

Código orientativo para la capa:

class DebtBurdenLayer(tf.keras.layers.Layer):
    def __init__(self, annuity_tau=0.5, credit_tau=5.0, epsilon=1e-6, **kwargs):
        super().__init__(**kwargs)
        self.annuity_tau = annuity_tau
        self.credit_tau = credit_tau
        self.epsilon = epsilon

    def call(self, inputs):
        income = inputs[:, 0:1]
        credit = inputs[:, 1:2]
        annuity = inputs[:, 2:3]

        annuity_income_ratio = annuity / (income + self.epsilon)
        credit_income_ratio = credit / (income + self.epsilon)

        annuity_ratio_sat = tf.math.tanh(annuity_income_ratio / self.annuity_tau)
        credit_ratio_sat = tf.math.tanh(credit_income_ratio / self.credit_tau)

        return tf.concat([annuity_ratio_sat, credit_ratio_sat], axis=1)

    def get_config(self):
        config = super().get_config()
        config.update({
            "annuity_tau": self.annuity_tau,
            "credit_tau": self.credit_tau,
            "epsilon": self.epsilon,
        })
        return config
