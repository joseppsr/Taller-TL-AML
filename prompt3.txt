Punto 2: FAIR Loss
==================
Implementa una función de pérdida customizada basada en:

FairLoss = BinaryCrossentropy(y_true, y_pred) + lambda_fair * corr(y_pred, s)^2

Donde:
- `y_true` es la variable objetivo `TARGET`.
- `y_pred` es la probabilidad predicha de dificultad de pago.
- `s` es la variable sensible `CODE_GENDER` codificada como 0/1.
- `lambda_fair` controla la fuerza de la penalización FAIR.
- `corr(y_pred, s)^2` penaliza tanto dependencia positiva como negativa y fuerza la correlación hacia cero.

Importante:
===========
Keras llama a la loss como `loss(y_true, y_pred)`, por lo que la variable sensible debe pasarse dentro de un `y_true` extendido.

Crea:

```python
y_train_fair = np.column_stack([y_train_np, s_train_np]).astype("float32")
y_val_fair   = np.column_stack([y_val_np, s_val_np]).astype("float32")

```

Dentro de la loss:

```python
y_true_label = y_true[:, 0:1]
sensitive    = y_true[:, 1:2]
```

La función debe calcular:

1. Binary crossentropy entre `y_true_label` y `y_pred`.
2. Correlación de Pearson entre `y_pred` y `sensitive` dentro del batch.
3. Penalización `corr ** 2`.
4. Pérdida total: `bce + lambda_fair * corr_squared`.

=======================
Añade una clase de pérdida similar a esta, adaptándola al código del notebook:

```python
class FairCorrelationLoss(tf.keras.losses.Loss):
    def __init__(self, lambda_fair=0.1, name="fair_correlation_loss"):
        super().__init__(name=name)
        self.lambda_fair = lambda_fair
        self.bce = tf.keras.losses.BinaryCrossentropy()

    def call(self, y_true, y_pred):
        y_true_label = y_true[:, 0:1]
        sensitive = y_true[:, 1:2]

        y_pred = tf.cast(y_pred, tf.float32)
        y_true_label = tf.cast(y_true_label, tf.float32)
        sensitive = tf.cast(sensitive, tf.float32)

        bce_loss = self.bce(y_true_label, y_pred)

        y_pred_centered = y_pred - tf.reduce_mean(y_pred)
        sensitive_centered = sensitive - tf.reduce_mean(sensitive)

        numerator = tf.reduce_sum(y_pred_centered * sensitive_centered)
        denominator = tf.sqrt(
            tf.reduce_sum(tf.square(y_pred_centered)) *
            tf.reduce_sum(tf.square(sensitive_centered))
        ) + tf.keras.backend.epsilon()

        corr = numerator / denominator
        corr_squared = tf.square(corr)

        return bce_loss + self.lambda_fair * corr_squared

    def get_config(self):
        config = super().get_config()
        config.update({"lambda_fair": self.lambda_fair})
        return config
```

Notas técnicas:
- Usa `corr_squared`, no `corr`, porque queremos llevar la dependencia a cero, no hacerla negativa.
- Añade `epsilon` en el denominador para evitar divisiones por cero.
- Cuida las dimensiones `(n, 1)` para evitar errores de broadcasting.
